# Incident Report

## Purpose

Use when incident response work is transferred to recovery, review, or follow-up owners.

## Template

- Summary:
- Current Status:
- Completed Work:
- Pending Work:
- Risks:
- Required Documents:
- Required Reviewers:
- Recommended Next Step:

## Example

- Summary: Auth login flow implementation is complete and ready for QA validation.
- Current Status: waiting for integration and regression review
- Completed Work: web login UI, auth service contract updates, registry updates
- Pending Work: release validation and observability check
- Risks: MFA extension not included in current scope
